# KarimunTrip - Template Website Travel HTML Bootstrap

Template static website travel/paket wisata menggunakan HTML, CSS custom, Bootstrap 5, dan sedikit JavaScript.

## Isi File
- index.html = Beranda
- paket.html = Daftar paket wisata
- detail-paket.html = Detail paket
- destinasi.html = Daftar destinasi
- jadwal-kapal.html = Jadwal kapal/trip
- galeri.html = Galeri foto
- kontak.html = Form konsultasi WhatsApp
- assets/css/style.css = CSS utama
- assets/js/main.js = Script tahun otomatis dan form WhatsApp

## Cara Membuka
Cukup klik dua kali `index.html` di browser.

## Cara Edit Nomor WhatsApp
Cari dan ganti semua nomor berikut:
6281234567890
0812-3456-7890

## Cara Ganti Nama Brand
Cari teks:
KarimunTrip
lalu ganti dengan nama brand klien.

## Cara Ganti Foto
Foto saat ini memakai link Unsplash online di `assets/css/style.css` dan beberapa tag `<img>` di `galeri.html`.
Untuk project real, disarankan download foto asli klien lalu simpan di folder `assets/img/`.

## Catatan
Template ini dibuat sebagai UI static. Belum ada backend/admin CMS.
